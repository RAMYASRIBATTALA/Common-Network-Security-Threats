# Common-Network-Security-Threats

This task involves writing a report on common network security threats like DoS, MITM, and spoofing. It explains what these attacks are, how they work, their impact on systems, and methods to prevent them. The goal is to understand basic cyber threats and improve network security awareness.


## Overview

This project presents a detailed study of common network security threats and their countermeasures. It focuses on understanding attacks such as Denial-of-Service (DoS), Man-in-the-Middle (MITM), and Spoofing, along with their working, impact, and prevention techniques.



##  Objective

* To understand common network security threats
* To analyze how these attacks work
* To study their impact on systems and networks
* To identify effective prevention methods



##  Network Security Basics

Network security is the process of protecting systems and data from unauthorized access and cyberattacks. It ensures:

* **Confidentiality** – Data is accessed only by authorized users
* **Integrity** – Data is not altered without permission
* **Availability** – Systems remain accessible



##  Common Network Security Threats

### 1️ Denial-of-Service (DoS) Attack

**Description:**
A DoS attack floods a server with excessive traffic, making it unavailable to legitimate users.

**How it Works:**

* Attacker sends large number of requests
* Server becomes overloaded
* Service becomes unavailable

**Impact:**

* Website downtime
* Loss of users and revenue

**Prevention:**

* Firewalls
* Rate limiting
* Load balancing



### 2️ Man-in-the-Middle (MITM) Attack

**Description:**
An attacker secretly intercepts communication between two parties.

**How it Works:**

* Attacker positions between sender and receiver
* Data is intercepted or modified

**Impact:**

* Data theft
* Privacy breach

**Prevention:**

* Use HTTPS
* Avoid public Wi-Fi
* Use VPN
* Enable two-factor authentication



### 3 Spoofing Attack

**Description:**
Spoofing is when an attacker pretends to be a trusted source.

**Types:**

* IP Spoofing
* Email Spoofing
* ARP Spoofing

**Impact:**

* Unauthorized access
* Data theft

**Prevention:**

* Authentication methods
* Email filtering
* User awareness




##  General Security Measures

* Use strong passwords
* Keep software updated
* Install antivirus and firewall
* Avoid unsecured networks
* Monitor system activity


##  Conclusion

Network security threats such as DoS, MITM, and spoofing can cause serious damage to systems and data. Understanding these attacks and applying proper preventive measures can help protect networks and ensure secure communication.
